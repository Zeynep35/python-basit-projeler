# Not Defteri
Basit bir komut satırı not uygulaması. Kullanıcı not ekleyebilir, silebilir, güncelleyebilir veya listeleyebilir. Bu projedeki amacımız aslında python ile dosya işlemlerini yapabilmek. İlk versiyonu eklendi. Geliştirilmeye devam edilecek.

## Nasıl çalıştırırlır.
```bash
python app.py

```

## Veriler
Tüm notlar `notlar.json` dosyasına kaydedilir. Uygulama bu dosyayı otomatik olarak oluşturur.

Hoşgeldiniz....
