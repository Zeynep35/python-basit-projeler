# Taş/Kağıt/Makas Oyunu
Bu proje, kullanıcı ile bilgisayar arasında oynanan basit bir Taş / Kağıt / Makas oyunudur.
Oyun 3 turdan oluşur. Her turda kullanıcı ve bilgisayar bir seçim yapar, sonuçlar karşılaştırılır ve 3 turun sonunda nihai kazanan ekrana yazdırılır.

## Nasıl çalıştırılır.
```bash
Aşağıdaki komutu terminalde çalıştırarak oyunu başlatabilirsiniz:
python app.py 


Oyun Kuralları

Taş, makas'ı yener

Makas, kağıt'ı yener

Kağıt, taş'ı yener

Oyun 3 turdan oluşur, her turun sonunda kazanan belirlenir.
Üç turun sonunda toplam puanlara göre nihai kazanan açıklanır.

## Örnek Çıktı

seçtiğiniz taş: kağıt, bilgisayarın seçimi: makas
Bilgisayar kazandı.
            2. Tur
['taş', 'kağıt', 'makas']taş
seçtiğiniz taş: taş, bilgisayarın seçimi: makas
Tebrikler kazandınız..!
            3. Tur
['taş', 'kağıt', 'makas']makas
seçtiğiniz taş: makas, bilgisayarın seçimi: taş
Bilgisayar kazandı.
Sonuç: Siz: 1 - Bilgisayar: 2

```
