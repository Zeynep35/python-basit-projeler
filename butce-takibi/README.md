# Bütçe Hesaplama Uygulaması
Basit bir terminal uygulaması olarak aylık bütçenizi ve giderlerinizi takip etmenizi sağlar.

# Gereksinimler
- Python 3.8 veya üzeri

# Nasıl çalıştırılır.
```bash
python app.py

```

# Örnek Çıktı

Hadi bütçemizi düzenleyelim...
Aylık toplam bütçenizi giriniz lütfen: > 1000
DEBUG: <class 'int'> 1000

-----------------------------
1. Bütçeyi Güncelle
2. Gider Ekleyiniz
3. Gider Hesaplayınız
4. Kalan Paranızı Görün
5. Çıkış
-----------------------------

Yapmak istediğiniz işlemi seçiniz: > 1
Gider ismini yazınız lütfen: > elma
Bu ürünün tutarı ne kadar?: > 20
Gider eklendi: elma (20 TL)
