## Zar Atma Oyunu
Basit bir zar oyunudur. 1-6 aralığında bilgisayar otomatik bir sayı belirler. 
Sonucu ekrana yazar. Eğer kullanıcı oyuna devam etmek isterse program zar atmaya devam eder. 

## Nasıl çalıştırırlır.
```bash
python app.py
