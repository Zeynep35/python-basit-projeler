# Banka Uygulaması
Python ile temel dosya işlemleri ve koşul yapıları kullanılarak geliştirilmiş basit bir banka simülasyonu.

## Nasıl çalıştırılır?
```bash
python app.py

```

## Örnek Çıktı


X bankasına hoşgeldiniz
Gönderici ismini yazınız: A
Alıcı isminizi yazınız: B
Göndermek istediğiniz tutarı giriniz: 500

            1. Bakiye gör
            2. Para transferi yap
            3. Çıkış

Yapmak istediğiniz işlemi seçiniz: (1-3): 1
Kullanıcı id'inizi giriniz: (A/B)A
Toplam bakiyeniz: 1000 TL

            1. Bakiye gör
            2. Para transferi yap
            3. Çıkış

Yapmak istediğiniz işlemi seçiniz: (1-3): 2
Gönderici id isminizi yazınız: (A/B)A
Alıcı isminizi yazınız: (A/B)B
Göndermek istediğiniz tutarı giriniz: 300
Transfer işlemi başarılı gerçekleşti. Kalan bakiyeniz: 700

            1. Bakiye gör
            2. Para transferi yap
            3. Çıkış

Yapmak istediğiniz işlemi seçiniz: (1-3): 3
